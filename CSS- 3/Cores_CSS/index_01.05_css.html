HSL

hsl(hua, saturation , lightness)
  • Hue (matiz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • saturation (saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
