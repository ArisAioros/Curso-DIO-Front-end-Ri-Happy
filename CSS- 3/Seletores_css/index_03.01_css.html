Agrupamento de Seletores

Podemos aplicar as memas regras CSS para vários seletores diferentes. Para isso podemos separar os seletores atilizando uma vigula (,)

Assim, não precisamos ficar copiando e colando os mesmos códigos paa todos, nós podemos fazer um só bloco de CSS e reulizar as mesmas regras!
